# rna-structural-prediction
GitHub Repository for Project: RNA Structure Prediction: A Unified Benchmark from Dynamic Programming to DCSE-6301-Aeep Learning for the Course 
This repository contains the **final submission** for an RNA secondary structure prediction benchmark project.  
It implements and evaluates a **Numba-accelerated Nussinov dynamic programming baseline** on two standard RNA structure datasets under a unified decoding and evaluation framework.

This submission focuses exclusively on the **classical baseline** and its full-dataset evaluation.
